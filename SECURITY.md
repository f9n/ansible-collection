# Security Policy

## Supported versions

Only the latest released version of this collection receives security fixes.

| Version | Supported |
| ------- | --------- |
| latest  | yes       |
| older   | no        |

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

To report a vulnerability, open a [GitHub Security Advisory](https://github.com/f9n/ansible-collection/security/advisories/new)
or contact the maintainers privately via GitHub.

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected versions
- Any suggested mitigations

You will receive a response within 7 days. If the issue is confirmed, a fix will
be released as soon as possible and credited to the reporter (unless anonymity is requested).
